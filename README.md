
Laravel 9 Crud App
## How To Run The Project Laravel

Requirements 
1. Laravel 9 
Mysql DB

- **[Mysql Steps].**

1. Mysql -u `username` -p `password`
2. Crearte Database `dbName`

- **[Environment].**
Edit the .env.example and fill in your database infornation


- **[Migrating the Projects Database].**

1. `php artisan mimgrate`

- **[Running the Project].**

1. `php artisan serve`
